## Todo App with Redux Toolkit, LocalStorage, TailwindCSS

This is a React todo application that utilizes Redux Toolkit for state management, localStorage for persistence, TailwindCSS for styling, and React Icons for visual elements.

### Features

- Add, remove, and mark todos as completed.
- Todos persist across page refreshes using localStorage.
- Uses Redux Toolkit for efficient state management.
- Styled with TailwindCSS for a clean and responsive design.
- Leverages React Icons for visually appealing Todos.

### Installation

1. Clone this repository or download the zip file.
2. Install dependencies:

   ```bash
   npm install
   ```

### Usage

1. Run the development server:

   ```bash
   npm run dev
   ```

   This will start the application at http://localhost:5173/ by default.

2. Open the URL in your browser and start adding todos!

## Built-in Functionality

- Clicking the checkbox toggles the completion status of a todo.
- Clicking the "Remove" button removes the corresponding todo.
- Pressing Enter in the input field adds a new todo.
- The "Add Todo" button also adds a new todo with the provided text.

## Technologies Used

- React: JavaScript library for building user interfaces.
- Redux Toolkit: Simplified implementation of Redux for state management.
- localStorage: Web Storage API for persisting data locally.
- TailwindCSS: Utility-first CSS framework for rapid styling.
- React Icons: Library of pre-built SVG icons for React projects.
